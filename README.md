## LinkTree!

A personal link-in-bio page with animated gradient text and social media connections.

```
# Features:
- Animated orange-to-pink gradient text with glow effect
- Time-based greeting (morning/afternoon/evening/night)
- Social media links with gradient icons
- Dark theme with ambient particle background
- Responsive design for all screen sizes
- Smooth hover animations on cards

# Social Links Included:
- GitHub
- Instagram
- Pinterest
- Discord

# Technologies Used:
- HTML5
- CSS3 (custom properties, animations, gradients)
- Vanilla JavaScript
- Google Fonts

# Setup:
1. Clone the repository
2. Open index.html in your browser
3. No build tools or dependencies required

# Customization:
1. Edit the href attributes in the card links
2. Update the handle text in each card
3. Modify the gradient colors in the CSS
4. Change the tagline text as needed

License: MIT
```
